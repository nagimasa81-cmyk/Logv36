LogMergeTool v34 Multi Log Viewer

追加内容:
- Log Viewerを 1 log / 2 logs (Dual) / 3 logs / 4 logs の表示モードで切替可能にしました。
- デフォルトは 2 logs (Dual) です。
- 各ログペイン上部に Log Type / Load This / Search / Noise Rule 操作を配置しました。
- Load Visible Logs で、現在表示されているペインだけをまとめて読み込みます。
- クリックした行のTimestampを基準に、他の表示ペインすべてで最も近い行へジャンプします。
- 中央/各ペイン間の境界はドラッグで左右に調整できます。
- Timestamp列は固定幅、Message列はペイン幅に合わせて伸縮します。
- Viewer Time Rangeは Focused pane または Visible panes へ適用できます。
- 選択行のExportは表示ペインの選択行に対応します。

使い方:
1. Open Dual Log Viewer / Log Explorer を開く。
2. View modeで 1/2/3/4 logs を選択。
3. 各ペインで表示したいログ種別を選択。
4. Load Visible Logsを押す。
5. 任意の行をクリックすると、他ペインが近いTimestampへジャンプします。
