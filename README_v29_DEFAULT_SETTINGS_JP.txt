LogMergeTool NoExcel v29 Default Settings / Tooltips

変更点:
- Reset to Defaultsボタンを追加。
- デフォルト設定:
  - Folders: 空白
  - Date Range: Start/Endとも未使用
  - Site Information: 空白
  - Log Types: すべて未選択
  - Performance: 高速推奨設定ON
  - Smart Noise: Apply approved Noise Rules in Merge output / Learning Modeの2つをON
- 通常起動時は前回の状態を復元。
- Performance項目にマウスオーバー説明を追加。
- Smart Noise項目にマウスオーバー説明を追加。
- 日付範囲、ログタイプ、Performance、Smart NoiseもQSettingsへ保存・復元。

使い方:
1. 通常は前回終了時の設定が復元されます。
2. 初期状態へ戻したい場合は Reset to Defaults を押します。
3. Performance / Smart Noise の各項目へマウスを置くと説明が表示されます。
